We build Web apps and Web browsers to connect cultures.

Building the Web

We write the specifications that define the Web, the C++ and Rust code that makes Web browsers, the tools used by Web developers, and Web apps that harness the unique capabilities of the Web.

Connecting cultures

We create apps to help individuals understand and engage with other cultures, and we encourage and amplify the contributions of individuals and corporations in Asia to shape the Web and the technology that affects our world.

About Birchill

Founded in July 2019 in Tokyo, Japan by current and former Web browser developers, we hope to continue building the Web from Asia. Through this we long to see individuals delight in crossing cultural boundaries, and to help companies enjoy new successes through deeper cross-cultural cooperation.
