building the Web

私たちは、Rust や C++ 言語によるウェブブラウザや ウェブ開発者のためのツール開発、 ウェブ標準仕様策定、 ウェブならではの機能を持つアプリケーション作成など、 ウェブの発展に貢献しています。

learning the world

私たちは、 世界の文化や世界を変えていく技術などに関与する 個々人の理解を深めるアプリケーションを作成しています。 また、真にグローバルなプラットフォームとしてのウェブの実現に向け、 アジアにおいて広くウェブへの貢献を促し、これをサポートします。

birchill(バーチル)について

株式会社 Birchill は、ウェブブラウザ開発者によって2019年6月東京に設立し、 アジアからウェブを作り続けています。

社名は丘（ヒル）の上にある白樺（バーチ）の木のイメージに由来しています。 日本で樺の木は、光、豊かさ、忍耐、優しさを表すと言われています。 急速に変化する業界の中で、 深い経験と技術力を活かし冷静かつ成熟した存在になることを目指しています。
